Getting started

  • node.js installed?
  • check the version:
    • $ node --version

install the YUIDOC package

  • $ npm -g install yuidocjs

FORK THIS Git repo, clone locally

  • git.corp.yahoo.com/aunz-webdev/yui-modules

cloned?

  • navigate to the top level folder:
  • $ cd /blah/.../multipass/yui-modules/

LET'S BUILD!

  • yui-modules$ yuidoc .

PRESTO!

  • the YUIDOCs will be in the 'doc' subfolder
  • If it's setup in your localhost, you can view it locally:
  • modernpast-lm...yui-modules/doc/

SERVER MODE

  • yui-modules$ yuidoc . --server 5000
  • watches for file changes
  • http://127.0.0.1:5000/
